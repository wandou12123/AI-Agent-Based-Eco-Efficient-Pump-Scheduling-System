# 泵站能效优化调度系统 - 前端

Vue 3 + TypeScript + Vite + Element Plus + ECharts

## 开发

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # 生产构建
```

## 页面

| 路由 | 组件 | 说明 |
|------|------|------|
| `/login` | LoginView | 登录/注册 |
| `/chat` | ChatView | AI 对话 (文本 + 语音 + 文书上传) |
| `/schedule` | ScheduleView | 调度任务管理 |
| `/station` | StationView | 泵站监控面板 |
| `/config` | ConfigView | 系统配置 |
